Please run Demo.m to compare the results of Non-Local Means (NLM) and Pruned Non-Local Means (PNLM).

List of the functions used in Demo.m
a) nlm.m : computes NLM.
b) pnlm.m : computes PNLM 
c) pnlm_sure.m : computes denosed image for each particular values of pruning parameter lambda.
